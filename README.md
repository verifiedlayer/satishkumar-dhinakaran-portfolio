# Satish Kumar Dhinakaran — Portfolio

Personal portfolio site for Satish Kumar Dhinakaran, Co-Founder & CBO of
[VerifiedLayer.ai](https://verifiedlayer.ai) — the API-first trust
infrastructure that powers how LLMs cite, evaluate, and recommend software.

## Tech Stack

- [React 19](https://react.dev/) + [React Router](https://reactrouter.com/)
- [Vite](https://vitejs.dev/)
- [Tailwind CSS 4](https://tailwindcss.com/) + [daisyUI](https://daisyui.com/)

## Getting Started

1. Clone the repository

```bash
git clone https://github.com/verifiedlayer/satishkumar-dhinakaran-portfolio.git
```

2. Install dependencies

```bash
npm install
```

3. Run the development server

```bash
npm run dev
```

4. Build for production

```bash
npm run build
```

## Deployment

```bash
npm run deploy
```

Builds the site and publishes the `dist` folder to GitHub Pages via
`gh-pages`.
